# Configure Header Settings

## **Header Panel**

To explore further settings, navigate to the following path based on your version:

=== "v4.5.1 and Later"

    !!! success "RealHomes Settings"
        <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">RealHomes</span> ➤ <span class="nav-step">Settings</span> ➤ <span class="nav-step">Site Identity</span> or <span class="nav-step">Headers</span>

=== "v4.5.0 and Earlier"

    !!! info "Legacy Settings"
        <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">RealHomes</span> ➤ <span class="nav-step">Customize Settings</span>

    ![Configure Header Settings](images/home-setup/header_settings.png)

Explore **Site Identity** and **Headers** section to explore further settings.

## Create Custom Header

You can create your own Custom Header using Elementor (free version) with RealHomes. You can consult <strong><a href="https://realhomes.io/documentation/custom-header-footer-elementor/">this section of the documentation</a></strong> to learn how.

## **Header Settings**

You can customize almost all the components of the header from this section.

![Configure Header Settings](images/home-setup/header_classic.png)

### **Basics**

Navigate based on your version:

=== "v4.5.1 and Later"

    !!! success "RealHomes Settings"
        <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">RealHomes</span> ➤ <span class="nav-step">Settings</span> ➤ <span class="nav-step">Headers</span> ➤ <span class="nav-step">Basic</span>

=== "v4.5.0 and Earlier"

    !!! info "Legacy Settings"
        <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">RealHomes</span> ➤ <span class="nav-step">Customize Settings</span> ➤ <span class="nav-step">Header</span> ➤ <span class="nav-step">Basics</span>

    ![Header Variation](images/header/header-settings-basics-classic.png)

### **Sticky Header**

The **Sticky Header** option is located in the **Basic** section mentioned above.

!!! info "Important"

    If you are using Custom Elementor based Header, then <a href="https://realhomes.io/documentation/custom-header-footer-elementor/#how-to-make-elementor-header-sticky">follow the instruction here</a> to make it sticky.

### **Variations**

To change header variation please navigate to the **Basic** section mentioned above and select **Center** for the option which says Choose **Header Variation**.

### **Header Styles**

To change header styles for Classic Design please navigate based on your version:

=== "v4.5.1 and Later"

    !!! success "RealHomes Settings"
        <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">RealHomes</span> ➤ <span class="nav-step">Settings</span> ➤ <span class="nav-step">Styles</span> ➤ <span class="nav-step">Header</span>

### **Responsive Header Styles**
