# Dashboard Labels

In order to change the Dashboard Menu labels, navigate based on your version of the **RealHomes** theme:

=== "v4.5.1 and Later"

    !!! success "RealHomes Settings"
        <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">RealHomes</span> ➤ <span class="nav-step">Settings</span> ➤ <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">Menu Labels</span>

    ![Dashboard Menu Custom Labels](images/dashboard/dashboard-menu-custom-label-settings.png)

=== "v4.5.0 and Earlier"

    !!! info "Legacy Settings"
        <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">RealHomes</span> ➤ <span class="nav-step">Customize Settings</span> ➤ <span class="nav-step">Dashboard</span> ➤ <span class="nav-step">Dashboard Menu Custom Labels</span>

    ![Dashboard Styles](images/dashboard/dashboard-menu-custom-labels.gif)

In this section, you can modify both page and menu titles for each section of the frontend user dashboard.
