# RealHomes Cloud Studio

RealHomes Cloud Studio is a powerful template library built directly into the RealHomes theme. It provides you with access to beautiful, pre-designed Elementor templates including headers, footers, full pages, and individual sections that you can import directly from the cloud with a single click.

This feature is designed to rapidly speed up your workflow and help you achieve stunning, professional designs without having to build every component from scratch.

---

## Key Features

**Smart Media Localization:**

When you import a template, RealHomes Cloud Studio doesn't just link to images hosted on external servers. It automatically downloads all external media (images, SVGs) and uploads them directly to your local **WordPress Media Library**. This ensures your pages load fast and your assets remain intact.

**Favorites System:**

Found a template you love? You can "Star" or favorite templates. <br>These are saved to your profile and easily accessible via a dedicated "Favorites" tab for quick access in future projects.

**Import History:**

Every time a template is imported, it is logged in the **Import History** (accessible via the WordPress Admin Bar or Dashboard). If you accidentally delete a section and need to find the exact template you used days later, you can easily retrieve it from your history.

**Advanced Filtering:**

Quickly find what you need by filtering templates by Category (e.g., Headers, Footers) and Style (e.g., Ultra, Modern), or by using the instantaneous search bar.

## Requirements

Before using RealHomes Cloud Studio, please ensure your setup meets the following requirements:

- **PHP Version:** 8.3 or higher (WordPress Requirement)
- **Elementor Page Builder:** Version 3.0.0 or higher
- **RealHomes Theme:** Must be installed and activated

!!! info "Important"

    If your environment does not meet these requirements, you will see a notice in your WordPress dashboard, and the Cloud Studio features will be disabled until the requirements are met.

## Accessing the Cloud Studio

You can browse and interact with the RealHomes Cloud Studio from two different places in your WordPress environment:

### 1. From the WordPress Dashboard
Navigate to **RH Cloud Studio** in your main WordPress admin menu. This brings up the standalone Template Library where you can browse the entire collection of designs available in the cloud.

## How to Import a Template

Importing a template is incredibly simple and takes only a few seconds:

1. **Open the Cloud Studio** either via the dashboard or directly inside the Elementor Editor.
2. **Browse or Filter** the library. You can filter templates by:
   - **Category:** (e.g., Headers, Footers)
   - **Type:** (e.g., Sections, Pages, Single Template, Listings Template)
   - **Style:** (e.g., Ultra, Modern, Classic)
3. **Preview** the template by clicking the **Preview** button to see a live demo of the design.
4. Once you find the design you like, click the **Insert** button.

The template, along with all its predefined styles, settings, and structural elements, will immediately be imported into your page.

## Syncing the Template Library

The RealHomes Cloud Studio library is constantly growing as we add new and exciting designs. 

To ensure you always have access to the latest templates, you can synchronize your local library with our cloud server:

1. Go to **RH Cloud Studio** in your WordPress dashboard.
2. Click the **Sync Templates** button (usually located near the top of the interface).
3. The system will connect to the cloud, fetch the latest manifest, and update your library with any newly added or updated templates.

!!! info "Tip"

    Make it a habit to click **Sync Templates** periodically to discover new designs added by the Inspiry Themes team!
