# ü§ù How to Set Up and Use the RealHomes CRM

The **RealHomes CRM** plugin brings a complete powerful all-in-one **Customer Relationship Management** solution to your website, helping you manage leads, track inquiries, and streamline your real estate communication process directly from your RealHomes Dashboard.

Follow the steps below to install, configure, and start using the RealHomes Customer Relationship Management system.

---

## üîå Step 1: Install & Activate the Plugin

1. Navigate to **Dashboard ‚Üí RealHomes ‚Üí Plugins** 
2. Find the **RealHomes CRM** there.
3. Click **Install** and then **Activate** the plugin

![Install RealHomes CRM Plugin](images/realhomes-crm/install-realhomes-crm-plugin.png)

Once activated, new CRM sections will appear. You will find **CRM Settings** under the **RealHomes** menu in your WordPress admin dashboard and a **Properties CRM** section in your frontend dashboard.

---

## ‚öôÔ∏è Step 2: Configure CRM Settings

Go to **Dashboard ‚Üí RealHomes ‚Üí CRM Settings** to configure the plugin's core options. The settings are organized into tabs.

### General Settings
**Empty Fields Fallback Value:** Define a fallback value (like '-') to display when a field is empty.

![CRM General Settings](images/realhomes-crm/crm-general-settings.png)

### Leads Settings
* **Name Prefixes:** Customize the list of titles for leads (e.g., Mr, Ms, Broker, Investor).
* **Lead Types:** Define the different categories for your leads (e.g., Prospect, Referral, Tenant).
* **Lead Statuses:** Set the pipeline stages for leads (e.g., New, Contacted, Qualified, Won, Lost).

![CRM Leads Settings](images/realhomes-crm/crm-leads-settings.png)

### Inquiries Settings
* **Inquiry Types:** Define the types of inquiries clients can make (e.g., Purchase, Rent, Valuation, Tour).
* **Inquiry Statuses:** Set the pipeline stages for inquiries (e.g., New, In Progress, Closed, Awaiting Response).

![CRM Inquiries Settings](images/realhomes-crm/crm-inquiries-settings.png)

### Deals Settings
* **Deal Stages:** Define the main stages for a deal (e.g., In Progress, On Hold, Won, Lost).
* **Deal Stages Colors:** Assign specific colors to each stage for quick visual identification.
* **Deal Probabilities:** Set predefined probability percentages (e.g., 10%, 50%, 90%).
* **Deal Last Steps:** Create a list of common "last actions" (e.g., Initial Contact, Proposal Sent).
* **Deal Next Steps:** Create a list of common "next actions" (e.g., Schedule Meeting, Send Contract).

![CRM Deals Settings](images/realhomes-crm/crm-deals-settings.png)

### Tasks Settings
* **Task Types:** Define the categories for tasks (e.g., Research, Meeting, Follow Up, Development).
* **Task Statuses:** Set the pipeline stages for tasks (e.g., Pending, In Progress, Completed, Deferred).
* **Task Priorities:** Customize the priority levels (e.g., Low, Medium, High, Urgent, Critical).

![CRM Tasks Settings](images/realhomes-crm/crm-tasks-settings.png)

## üß≠ Step 3: Understanding the CRM Dashboard

After setup, navigate to the **Properties CRM** section on your frontend dashboard to view and manage your data.

### RealHomes CRM Dashboard Overview

This screen provides a high-level overview, including:

* **Activities:** A feed of recent actions, such as new leads.
* **Leads Stats:** A count of leads from the last 24 Hours, Last Week, and Last Month.
* **Deals Stats:** A summary of deals by status (In Progress, On Hold, Won, Lost, Total).

![CRM Dashboard Overview](images/realhomes-crm/crm-dashboard.png)

### Leads

A list of all individual leads. You can manage and view **Person**, **Contacts** (Email/Phone), **Type**, **Status**, and **Source**.

![CRM Leads Dashboard](images/realhomes-crm/crm-leads-dashboard.png)

### Inquiries

A list of all property-specific inquiries. This table includes **ID**, **Lead**, **Type**, **Status**, **Property Type**, **Beds/Baths**, **Area**, and **Price**.

![CRM Inquiries Dashboard](images/realhomes-crm/crm-inquiries-dashboard.png)

### Deals

This is your sales pipeline. It shows a summary of deals by stage and a list including **Title**, **Lead Contact**, **Assigned Agent**, **Last Step**, **Next Step**, **Stage**, and **Probability**.

![CRM Deals Dashboard](images/realhomes-crm/crm-deals-dashboard.png)

### Tasks

A to-do list for managing CRM-related actions. The list includes **Task**, **Description**, **Status**, **Type**, **Priority**, and **Due Date**.

![CRM Tasks Dashboard](images/realhomes-crm/crm-tasks-dashboard.png)

---

## üß© Step 4: Accessing CRM from Frontend Dashboard

As shown in the screenshots, the primary interface for managing leads, inquiries, deals, and tasks is the **RealHomes Frontend Dashboard**.

From there, agents and users can:

- View all their assigned leads
- Track lead activity and communication
- Update lead status (New, Contacted, Closed, etc.)
- Add internal notes for collaboration

You can also use the **+ Add New**, **Import**, and **Export** buttons available on the Leads, Inquiries, Deals, and Tasks pages to manage your data.

![Frontend CRM Access](images/realhomes-crm/crm-add-new-leads.png)
